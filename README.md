# 수업기록 자동완성 툴

수학 강사로 일하면서 매일 반복하던 수업기록 작성을 자동화한 웹 툴입니다.

## 만든 이유

수업 종료 후 원장님께 카톡으로 일일 수업기록을 전송해야 했는데,
매번 메모장에 작성 후 형식에 맞게 복사하는 과정이 번거로워 직접 만들었습니다.

## 기능

- 날짜 / 출퇴근 시간 자동 입력 (수동 변경 가능)
- 초등 / 중 / 고 학년별 학생 등록 및 관리
- 수업할 학생을 클릭해서 선택 → 선택된 학생만 기록 입력
- 학생별 수업 내용 / 과제 / 과제 완료 여부 입력
- 결석 / 병결 체크 및 결석 사유 입력
- 카톡 전송 형식으로 자동 변환 후 복사
- 학생 목록 및 날짜별 수업 기록 브라우저에 자동 저장
